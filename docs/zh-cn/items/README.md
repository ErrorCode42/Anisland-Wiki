#### 术式符文

术式符文是包含特定功能法术的牛皮纸。可以通过镌刻在工具或者武器上，在使用时获得相应效果。有专门用于保存术式符文的背包工具「符文宝典」。

游戏机制中的「武器DIY」有专门介绍如何使用术式符文。
